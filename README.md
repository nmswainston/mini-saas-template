# Mini SaaS Template

A minimal, production-ready SaaS starter template — clone it and ship faster.

## Overview

A lightweight SaaS boilerplate with the essentials pre-wired: auth-ready structure, Tailwind styling, CSP configuration, and Netlify deployment. Built to be the starting point for small SaaS products without the bloat of heavier frameworks.

## Tech Stack

- TypeScript
- Vite
- Tailwind CSS
- Netlify (deployment)

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
src/         # Application source code
public/      # Static assets
CSP.md       # Content Security Policy documentation
```

## Deployment

Deploy to Netlify with one click — configuration is in `netlify.toml`. The `CSP.md` file documents the Content Security Policy setup.
