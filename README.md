# Mini SaaS Template

A minimal, production-ready SaaS starter — clone it and ship faster.

## Problem

Starting a new SaaS project from scratch means wiring up the same boilerplate every time: build config, Tailwind, CSP headers, deployment config. It slows down the time from idea to working product.

## Solution

A lean, opinionated starter template with the essentials pre-configured — Vite, Tailwind, Content Security Policy, and Netlify deployment — so you can skip straight to building features.

## Tech Stack

- TypeScript

## Features

- Pre-configured Vite + TypeScript build pipeline
- Tailwind CSS with PostCSS ready to go
- Content Security Policy documented and configured (`CSP.md`)
- One-click Netlify deployment via `netlify.toml`
- ESLint configured out of the box

## Installation

```bash
npm install
npm run dev
```

## Lessons Learned

- A well-structured template saves hours on every new project
- CSP configuration is often left until last — building it in from the start prevents headaches
- Opinionated defaults are better than options when the goal is speed
